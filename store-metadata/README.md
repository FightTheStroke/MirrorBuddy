# Store Metadata

This directory contains metadata files for MirrorBuddy app store submissions.

## Structure

- `ios/` - App Store (Apple) metadata
- `android/` - Play Store (Google) metadata
- `screenshots/` - Screenshots for both platforms (to be added)

## Required Screenshots

### iOS (App Store)

- **iPhone 6.7"** (iPhone 14 Pro Max, 15 Pro Max): 1290x2796 pixels
- **iPhone 6.5"** (iPhone 11 Pro Max, XS Max): 1284x2778 pixels
- **iPad 12.9"** (iPad Pro): 2048x2732 pixels

### Android (Play Store)

- **Phone**: 1080x1920 pixels (minimum)
- **Tablet**: 1200x1920 pixels (7-inch)

## Screenshot Guidelines

1. Show the app in action with real content
2. Highlight key features for children with disabilities
3. Demonstrate accessibility features clearly
4. Use high-contrast, readable text overlays
5. Include diverse representation of children
6. Show multiple language support

## App Information

- **Name**: MirrorBuddy
- **Developer**: FightTheStroke Foundation
- **Target Audience**: Children with disabilities (cerebral palsy, learning difficulties)
- **Age Rating**: 4+ (iOS), Everyone (Android)
- **Categories**: Education, Health & Fitness
- **Languages**: English, Italian, French, German, Spanish
