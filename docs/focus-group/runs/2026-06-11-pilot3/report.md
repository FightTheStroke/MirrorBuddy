4. **FGOP-11 (proposto)**: eseguire il prossimo pass con somministrazione multi-turno via SendMessage (canary + controllo positivo iniettati in corso, §6.2), Verificatore Opus separato, e completamento Parte 2 (T4, T5, misure §4.5).
5. **n=3, subset non rappresentativo**: mancano ADHD puro (P2), autismo (P5, la più giovane), sordità (P6), paralisi cerebrale (P7), genitore-proxy (P8). Le classi di finding legate a impulsività, letteralità, no-audio e catena adulto NON sono state osservate per costruzione.
