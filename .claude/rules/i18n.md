# i18n — MirrorBuddy

## ALL UI text i18n'd. 5 locales day-1.

**Locales**: `it` (default) | `en` | `fr` | `de` | `es`
**Framework**: next-intl (App Router, server-first)
**Path**: `messages/{locale}/{namespace}.json` (ADR 0082)
**Namespaces**: common, auth, admin, chat, home, tools, settings, compliance, consent, education, navigation, errors, welcome, metadata, pricing, marketing, achievements, maintenance, voice, analytics, waitlist

## Wrapper key (ADR 0104 — CRITICAL)

Every JSON wraps content under filename key: `{ "namespace": { ... } }`. Missing = namespace collisions (`Object.assign`) → 98 ESLint warn + 37 E2E fail (past incident).

## Enforcement

| Rule | How |
|---|---|
| No hardcoded text in JSX | ESLint `no-hardcoded-italian` |
| camelCase keys only (ADR 0091) | ESLint `no-kebab-case-i18n-keys` |
| All 5 locales synced | Pre-commit `npm run i18n:check` |

## Hooks

| Context | Hook |
|---|---|
| Client component | `useTranslations('ns')` |
| Server component | `getTranslations('ns')` |
| Multiple ns | `useTranslationsGlobal()` |
| Current locale | `useLocaleContext()` |

## Add keys: IT first → `npx tsx scripts/i18n-sync-namespaces.ts --add-missing` → `npm run i18n:check`
